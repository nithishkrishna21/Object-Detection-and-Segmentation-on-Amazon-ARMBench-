# Object-Detection-and-Segmentation-on-Amazon-ARMBench-
Amazon ARMBench (Object Detection and Segmentation)

This reporsitory contains the codebase and other relevant files for my Computer Vision Project. 

The prpject aims to deliver a robust pipeline for segmentating and detecting objects in cluttered robotic environment. We have employed the Amazon ARMBench dataset to train the models that we are testing. Due to compute constraints, we only trained our model on a public subset of the dataset. Despite the setback, we achieved promising results while evaluating the models, which leads us to believe that the models used can be deployed to other environments as well.
